# 🌙 Moonieverse

### Exploring Love, Emotions & Human Connections

Moonieverse is a static web platform that presents **relationship insights and love psychology** in a visually engaging and easy-to-understand format. It focuses on delivering meaningful content through a clean UI and smooth user experiences.

---

## 💡 About the Project

Moonieverse is designed to share **relationship facts, emotional insights, and psychological perspectives on love**. The goal is to create a space where users can explore human connections through thoughtful and relatable content.

---

## 🚀 Features

* ❤️ **Relationship Facts & Insights**
  Curated content focused on love and human psychology

* 🎨 **Clean & Minimal UI**
  Designed for readability and user comfort

* 📱 **Responsive Design**
  Works across different screen sizes

* ✨ **Interactive Elements**
  Smooth transitions and basic JavaScript interactions

---

## 🧑‍💻 Tech Stack

* **HTML** – Structure
* **CSS** – Styling and layout
* **JavaScript** – Interactivity and dynamic behavior
